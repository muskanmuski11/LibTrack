# 📚 LibTrack – Library Book Inventory System

 1. Overview

**LibTrack** is a Python-based command-line application designed to manage and organize a library's book inventory efficiently. It allows users to perform essential **CRUD (Create, Read, Update, Delete)** operations on book records while demonstrating core programming concepts and database integration using SQLite.

This project is ideal for beginners and aspiring data analysts looking to understand how structured data systems are built and managed.

---

2. Features

* ➕ Add new books to the inventory
* 🔍 Search books by **title, author, or ID**
* ✏️ Update existing book details
* ❌ Delete books from the system
* 📖 View all books in the inventory
* 💾 Persistent storage using **SQLite database (library.db)**

---

 3. Technologies Used

5. Installation & Setup

# Step 1: Clone the Repository

```bash
git clone https://github.com/your-username/libtrack.git
cd libtrack
```

# Step 2: Run the Application

```bash
python "library book tracker.py"
```

> Make sure Python is installed on your system (Python 3.x recommended).

 6. Usage

Once the program starts, users can interact with the system through a menu-driven interface:

* Add new books by entering details like title, author, etc.
* Search for books using keywords or IDs
* Update existing records
* Delete unwanted entries
* View all stored books in a structured format

The interface is simple and intuitive, making it easy to navigate.
